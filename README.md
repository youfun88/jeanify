# jeanify

Static marketing site for **Jean Riley** — San Diego Realtor®.
Pre-compiled React (no build step) + CSS + assets. Served on GitHub Pages.

## Deploy (CI/CD)
Push to `main` → `.github/workflows/deploy.yml` publishes to GitHub Pages.

## Local preview
```sh
python3 -m http.server 8000   # then open http://localhost:8000
```
